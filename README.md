# ProbizmoStudy

## Getting Started

1. Install Vagrant to your host OS.

  https://www.vagrantup.com/

2. Clone this project.

  ```
  git clone https://github.com/m-imaoka/ProbizmoStudy.git
  ```

3. Move current directory to ProbizmoStudy.

  ```
  cd [ProbizmoStudy's path]

  ```
4. Run Vagrant.

  ```
  vagrant up
  ```
